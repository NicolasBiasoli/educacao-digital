5 Extensões do VS Code que Todo Iniciante Precisa Usar

1. Prettier - Code Formatter

Quando estamos aprendendo, é normal o código ficar bagunçado, com espaços tortos e linhas desalinhadas. O Prettier resolve isso instantaneamente. Ele é um formatador automático de código. Basta salvar o arquivo (Ctrl + S ou Cmd + S) e a extensão organiza toda a indentação, quebras de linha e ponto-e-vírgula magicamente. Seu código fica limpo, legível e com padrão profissional.

2. Live Server

Se você está estudando desenvolvimento web (HTML, CSS e JavaScript), o Live Server é indispensável. Sem ele, toda vez que você faz uma alteração no código, precisa ir ao navegador e atualizar a página manualmente. Com essa extensão, você clica em um botão e ela cria um servidor local integrado. Qualquer mudança salva no VS Code reflete no navegador em tempo real, sem que você precise recarregar nada.

3. GitLens — Git supercharged

Aprender a usar o Git e o GitHub pode ser confuso no início. O GitLens ajuda a visualizar o histórico do seu código de forma simples. Ele mostra, diretamente na linha onde você está digitando, quem fez aquela alteração, quando e em qual commit isso aconteceu. É uma ferramenta fantástica para entender a evolução do seu próprio projeto ou para trabalhar em equipe sem se perder nas versões.

4. Error Lens

Por padrão, o VS Code mostra erros de sintaxe colocando uma linha ondulada vermelha bem sutil embaixo da palavra errada. Para iniciantes, esse aviso costuma passar despercebido. O Error Lens melhora isso drasticamente: ele destaca a linha inteira e escreve a mensagem de erro exata com letras grandes logo ao lado do código. Você descobre na hora se esqueceu de fechar uma chave { ou uma tag HTML.

5. Material Icon Theme

À medida que seus projetos crescem, a árvore de arquivos na barra lateral do VS Code vira uma floresta de textos sem graça. O Material Icon Theme substitui os ícones padrão por ilustrações coloridas e visualmente intuitivas para centenas de tipos de arquivos e pastas. Arquivos JavaScript ganham um ícone amarelo "JS", pastas de imagens ganham um design de câmera, e assim por diante. Isso acelera muito a sua navegação visual.
